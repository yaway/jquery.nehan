Install

Include css, js in the <head>.

Offcource, you need to include nehan.js before jquery.nehan.js.

<link href="/path/to/nehan.css" type="text/css" rel="stylesheet">
<script src="/path/to/nehan.js" type="text/javascript"></script>
<script src="/path/to/jquery.nehan.js" type="text/javascript"></script>

Getting Started

Here is minimus sample code.

$(".my-nehan-document").nehan({
  direction:"hori", // or "vert"
  hori:"lr-tb", // "rl-tb" is not supported.
  vert:"tb-rl", // or "tb-rl"
  fontSize:16,
  width: 640,
  height: 480
});

Document is horizontal, and page size is 640x480, and font size is 16px.

Reader ui is displayed at each $(".my-nehan-document").

Options

direction : String

vert or hori is enabled.

hori : String

Document mode when direction property is assigned to "hori".

Only "lr-tb" is supported.

Notice that "lr" means inline-text flows from left to right, and "tb" means block level line flows from top to bottom.

vert : String

Document mode when direction property is assigned to "vert".

Both "tb-rl" and "tb-lr" are supported.

Notice that "tb" means inline-text flows from top to bottom, and "rl" means block level line flows from right to left, and "lr" means left to right.

fontSize : Integer

Define basic font size in px size.

width : Integer

Define screen width of reader ui in px size.

height : Integer

Define screen height of reader ui in px size.

text : String

By default, source text is obtained from each jquery object.

But you can use direct text value using property text.

$(".my-nehan-document").nehan({
  text:"not from dom text, but direct text",
  direction:"hori", // or "vert"
  hori:"lr-tb", // "rl-tb" is not supported.
  vert:"tb-rl", // or "tb-rl"
  fontSize:16,
  width: 640,
  height: 480
});

vertFontFamily : String

font-famiy in vertical mode.

Default value is "'ヒラギノ明朝 Pro W3','Hiragino Mincho Pro','HiraMinProN-W3','IPA明朝','IPA Mincho', 'Meiryo','メイリオ','ＭＳ 明朝','MS Mincho', monospace".

horiFontFamily : String

font-famiy in horizontal mode.

Default value is "'Meiryo','メイリオ','Hiragino Kaku Gothic Pro','ヒラギノ角ゴ Pro W3','Osaka','ＭＳ Ｐゴシック', monospace".

rowCount : Integer

Row count of screen division. 1 or 2 available.

colCount : Integer

Col count of screen division. 1 or 2 available.

spacingSize : Integer

Space size of each divided page.

usePager : Boolean

If false, pager object not displayed, and each paged-media appended to previous one.

This is true by default.

useWheel : Boolean

Mouse wheel is enabled if true, true by default.

useNombre : Boolean

Nombre number is displayed if true, false by default.

readerElements : String Array

UI orders of reader.

By default, this is ["screen", "pager"].

pagerElements : String Array

UI orders of pager.

By default, this is ["left-next", "indicator", "right-prev"].

UI parts are combination of "left-next", "left-prev", "right-next", "right-prev", "indicator", "progress".

"indicator" consists of page_no and total_page_count.

"progress" consists of just page_perecnt.

samples of pager elements

["left-prev", "progress", "right-next"]
["left-prev", "right-next", "progress"]
["left-prev", "indicator", "right-next"]

onCreateEngine : function(engine)

Called when layout engine is created and ready to use.

$(".my-nehan-document").nehan({
  direction:"hori", // or "vert"
  fontSize:16,
  width: 640,
  height: 480,
  onCreateEngine : function(engine){
    engine.setStyle("h1", {"font-size":"30px"});
  }
});

onReadyPage : function(reader)

Called when first page is set to screen.

Usefull to setup keybord shortcut or enable variaous ui effects.

This is not called if usePager is false.

onPage : function(reader)

Called after page_no is updated.

This is not called if usePager is false.

onComplete : function(reader)

Called after all pages generated.

This is not called if usePager is false.

And this callback is not called when you select pager-ui "progress".

By default, we append outline at the bottom of reader.

If you don't want it, disable like this.

$(".my-nehan-document").nehan({
  direction:"hori", // or "vert"
  fontSize:16,
  width: 640,
  height: 480,
  onComplete : function(reader){
    // do nothing
  }
});

onError : function(reader)

Called when something wrong.
